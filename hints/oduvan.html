Como identificar todos os movimentos válidos em uma configuração de tabuleiro do Jogo dos Pinos?

Primeiro precisamos identificar todos os espaços vagos identificado pelo '.'

Para isso, use a variação 'enumerate' do laço for. É a maneira mais eficiente de identificarmos as coordenadas dos pontos. Veja a sintaxe abaixo: 

             
   for indice1, letra1 in enumerate(palavra, start=1):
       # indice1 vai retornar a posição da letra1 no momento da execução do 'for'
       # letra1 é a variável de iteração do 'for', funciona da maneira que já conhecemos
       # palavra é a palavra na qual vamos fazer as iterações, também funciona da maneira que já conhecemos
       # start é o número que começamos a contagem. Por padrão, start vale 0
       print(indice1, letra1) # use o print caso queira ver o funcionamento

E como verifico quantos movimentos são possíveis?

Para cada furo encontrado ('.'), verifique as quatro direções (acima, abaixo, esquerda, direita) buscando por um padrão de um pino adjacente seguido pelo pino que fará o movimento. Use loops e condicionais para verificar essas condições em todas as células aplicáveis do tabuleiro.
